# DreamStay Hotel Management System

A modern, responsive hotel management web application built with React, TypeScript, and Tailwind CSS. This application provides a seamless booking experience for guests with features like room browsing, booking management, dining reservations, and contact services.

## 🌟 Features

- **Room Management**: Browse and search available rooms with detailed information
- **Booking System**: Easy room reservation with booking confirmation
- **Dining Services**: Restaurant and bar reservation system
- **Responsive Design**: Mobile-first design that works on all devices
- **Modern UI**: Beautiful interface built with Tailwind CSS and Framer Motion
- **Type Safety**: Full TypeScript implementation for better code quality

## 🛠️ Tech Stack

- **Frontend**: React 19 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS v4
- **Routing**: React Router v7
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Code Quality**: ESLint with TypeScript support

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/prateek8318/Hotel-management.git
cd Hotel-management
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## 🚀 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
